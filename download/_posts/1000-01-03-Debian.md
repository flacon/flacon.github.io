## Debian and Linux Mint "Debian"

*Debian* and *Linux Mint "Debian"* users can use repository at [OpenSUSE Build Service](http://download.opensuse.org/repositories/home:/Sokoloff/).

Open up a terminal and input these commands:

{% for ver in page.versions %}
### Debian {{ver}}

```
echo 'Types: deb
Enabled: yes
URIs: http://download.opensuse.org/repositories/home:/Sokoloff/Debian_{{ver}}/
Suites: ./
Components:
Signed-By: /etc/apt/trusted.gpg.d/flacon.asc' | sudo tee /etc/apt/sources.list.d/flacon.sources

wget -q -O- http://download.opensuse.org/repositories/home:/Sokoloff/Debian_{{ver}}/Release.key | sudo dd of=/etc/apt/trusted.gpg.d/flacon.asc

sudo apt update
sudo apt install flacon
```

{% endfor %}
